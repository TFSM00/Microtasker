Microtasker

A simple kanban task manager with boards, columns and cards, each tied to a user

Built by Tiago Moreira

Frontend

The styling was mainly created using Bootstrap, with some custom changes with CSS and Javascript for conditional styling.

Backend

The backend is all in Python using the Flask web development library.

Deployment

The whole website is deployed using Docker Compose. The containers are built on trigger from Github Actions, looking for push actions.


Hosting

The website is hosted within my own Raspberry Pi 4 Model B (1.8GHz - 8GB RAM), using Nginx as reverse proxy.

Database

All data is stored in a container with a Postgresql DB, controlled by Python's SQL Alchemy ORM.


Password Protection

This website uses a very high standard of password protection. When you sign up on this website, your password is transformed into a hash and salted (read more). Every time you login, the password you entered is hashed and compared to the hash in the database. By using salting, even if your password is the same as someone elses in the database, the salting ensures the hashes are different. The SHA256 algorithm is used to create the hash.
